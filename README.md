# frito
`frito` is an image reconstruction method that builds on the [`dorito`](https://github.com/maxecharles/dorito) image reconstruction framework. 
Both `frito` and `dorito` are designed to interface with [`amigo`](https://github.com/LouisDesdoigts/amigo), an end-to-end differentiable forward-modelling pipeline for analysing data from the Aperture Masking Interferometer (AMI) observing mode of the *James Webb Space Telescope* (JWST).

`frito` is designed to use autoencoders to create a machine-learned initial distribution and latent space in order to deconvolve images from JWST/AMI.

Please refer to these publications for more detail on `amigo` and `dorito`: [amigo paper here](https://arxiv.org/abs/2510.09806), [dorito paper here](https://arxiv.org/abs/2510.10924).
